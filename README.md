Title: To-Do List Application

Description:
This is a simple To-Do List application built using Python and Tkinter. It allows users to add tasks, delete selected tasks, and mark tasks as completed.

Features:
- Add tasks to the to-do list.
- Delete selected tasks.
- Mark tasks as completed with a checkmark.
- Scrollable task list for easy navigation.

How to Use:
1. Run the script to open the application.
2. Click "Add Task" to enter a new task.
3. Select a task and click "Delete Selected Task" to remove it.
4. Select a task and click "Mark as Completed" to mark it as done.

Requirements:
- Python 3.10
- Tkinter
